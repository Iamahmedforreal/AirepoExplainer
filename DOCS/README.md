Simple project documentation

This folder contains brief, easy-to-read documentation for the main parts of the project:

- `backend.md` - explains the Python backend structure under `app/` and how the pipeline works.
- `frontend.md` - explains the React frontend under `clerk-react/` and its entry points.
- `webhooks.md` - describes webhook ingestion and processing flow (Clerk webhook).
- `comment_director.md` - guidance for adding comprehensive inline comments and where to place them.

Use these docs to onboard new engineers or to prepare a README for each component.
